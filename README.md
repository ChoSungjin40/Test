# 로컬로 받아올 때 방법
디렉토리에서 cmd 입력후
```bash
git clone {address}
```
# 서버로 올릴때 방법
vsc 에서 파일을 연 후에 글을 변경 한 후 
```bash
git add {파일명 or .(.은 all을 의미)} #준비단계 stage
git commit -m "나타낼 commit message 내용 기재 : 변경점 서술하는것이 좋음" #변경사항 commit
git push #업로드 push
```
